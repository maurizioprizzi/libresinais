# Como contribuir com o LibreSinais

Obrigado pelo interesse em contribuir! Este projeto é uma doação para a comunidade surda brasileira e toda ajuda faz diferença.

## Primeiros passos

1. Faça um **fork** deste repositório
2. Clone o fork no seu computador: `git clone https://github.com/SEU-USUARIO/libresinais.git`
3. Crie uma branch para sua contribuição: `git checkout -b minha-contribuicao`
4. Faça suas alterações e commite: `git commit -m "Descrição do que fiz"`
5. Envie para o GitHub: `git push origin minha-contribuicao`
6. Abra um **Pull Request** descrevendo o que você fez

## Áreas onde precisamos de ajuda

### 🐍 Desenvolvimento Python
- Pipeline de reconhecimento (câmera → MediaPipe → classificador)
- Integração com LibreOffice via UNO
- Interface gráfica (preview da câmera)
- Testes automatizados

### 🤖 Machine Learning
- Treinamento e otimização de modelos
- Coleta e preparação de datasets
- Experimentação com novas arquiteturas

### 🤟 Comunidade Surda
- Validação dos sinais reconhecidos
- Gravação de dados para o dataset
- Feedback sobre usabilidade
- Definição de sinais prioritários
- Consultoria linguística sobre Libras

### 📝 Documentação
- Tradução e revisão de textos
- Tutoriais e guias de uso
- Documentação técnica

### 🧪 Testes
- Testes em diferentes plataformas (Linux, Windows, Mac)
- Testes com diferentes câmeras e condições de iluminação
- Reporte de bugs

## Regras simples

- Seja respeitoso com todos os contribuidores
- Use português nos comentários e documentação (código em inglês é ok)
- Descreva suas alterações de forma clara nos commits e pull requests
- Se for uma mudança grande, abra uma **issue** antes para discutirmos

## Dúvidas?

Abra uma **issue** no repositório com a tag `dúvida` e teremos prazer em ajudar.
