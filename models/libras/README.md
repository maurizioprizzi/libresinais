# Modelos treinados para Libras

Esta pasta conterá os modelos de Machine Learning treinados:

- `libras_alfabeto.tflite` — Modelo para o alfabeto manual (A–Z, Ç, 0–9)
- `libras_sinais.tflite` — Modelo para sinais dinâmicos (Fase 2)
- `classes.json` — Mapeamento de classes para texto

> Os modelos serão adicionados conforme o treinamento avançar.
